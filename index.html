Features

Everything your AI team needs

Neo brings together project management, agent orchestration, and real-time collaboration in a single native macOS window.

Multi-Agent Teams

Build teams from role-based templates — Developer, Designer, Analyst, Supervisor. Each agent has its own LLM provider and model.

Real-Time Events

Watch agents think, speak and go idle in real time via WebSocket. Status badges and activity feed update live as your team works.

Project Management

Organise work into projects. Each project has its own team, task history, and persistent WebSocket channel.

Natural Task Input

Type a task in plain English. The supervisor agent delegates to the right team members and coordinates the response automatically.

Native macOS Security

Built with App Sandbox and Hardened Runtime. Network access is scoped to localhost by default — your data never leaves your machine.

3D Workspace Coming soon

A Unity Metal view will replace the placeholder workspace — a living 3D office where your agents walk, talk and collaborate visually.

Live Workspace

Your agents, always in motion

Watch your team think, speak and collaborate in real time. Every task is broken down, delegated and completed — automatically.

How it works

From task to result in seconds

Neo's LangGraph supervisor pattern routes tasks to the best agent, collects results, and streams updates back to you.

01

Create a project & team

Add a project, then pick agents from role templates. Mix OpenAI, Anthropic, and Ollama models in the same team.

→
02

Send a task

Type your task in the workspace input. The supervisor decomposes it and assigns subtasks to the right agents.

→
03

Watch it happen live

Agent status badges flip to Thinking → Speaking → Idle. The activity feed shows every message as it arrives over WebSocket.

→
04

Get the result

The supervisor aggregates all agent outputs and delivers a final answer. Task complete.

Tech stack

Built on rock-solid foundations

macOS App
  • SwiftUI Native macOS 14+ UI
  • Observation @Observable state
  • URLSession async/await networking
  • WebSocket Live event stream
  • Unity 3D Metal view (coming)
Backend
  • FastAPI Async REST + WebSocket
  • LangGraph Supervisor agent pattern
  • LangChain OpenAI · Anthropic · Ollama
  • SQLAlchemy Async ORM + Alembic
  • SQLite / Postgres Dev & prod DB
